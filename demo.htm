Numbres & DateTime Counter

Number Counter:

 (3.5s)  (3s)  (5s)  (2.55s)  (2s)
  • jQuery is Required!
  • css file not used in section
  • add .fa-num-counter to your element (the name of this class is suggested)
  • add data-intravel="" on .fa-num-counter to set intravel time (second)
  • add data-counter-to="" on .fa-num-counter to set number of target
  • add .en-digit-counter on .fa-num-counter to En Digits
  • use $('.fa-num-counter').initNumberCounter(callback) to init this section, or $('.fa-num-counter').each((i. item)=> {$(item).initNumberCounter(callback)}) to init all sections
  • add callback function to execute when end counter
  • callback function have one argument which refers to .fa-num-counter

DateTime Counter Down To x:

  • jQuery is Required!
  • add .fa-dt-counter to your element and define 4 child elements for days, hours, minutes, seconds on .fa-dt-counter
  • add .fa-dt-counter-days on days, .fa-dt-counter-hours on hours, .fa-dt-counter-minutes on minutes, .fa-dt-counter-seconds on seconds
  • use data-dateTime="" on .fa-dt-counter to set Date or DateTime
  • use .en-digit-counter on .fa-dt-counter to En Digits
  • css file optional!
  • use data-content="" on .fa-dt-counter-days, .fa-dt-counter-hours, .fa-dt-counter-minutes, .fa-dt-counter-seconds to set name of section (css file Required)
  • css file, add ':', name of sections and don't allow moved section when counter (fix width)
  • use $('.fa-dt-counter').initDateCounters(callback) to init this sections
  • add callback function to execute when end counter
  • callback function have one argument which refers to .fa-dt-counter
